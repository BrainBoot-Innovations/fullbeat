Pass / Fail Criteria

GO for Production: All 30 Use Case tests pass. No more than 5 Negative failures with workarounds.
NO-GO: Any Use Case failure blocks production.
Bug Reporting: FAIL → auto Bug ID: Dhivya=T01-001, Divish=T02-001, Mugilan=T03-001. Add screenshots + steps to reproduce.
Test Accounts: Each tester registers their own account at /auth/register. Admin tests executed by Project Owner.
